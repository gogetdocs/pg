## UUID Type

The `uuid` data type in PostgreSQL stores Universally Unique Identifiers (UUID) as defined by RFC 4122, ISO/IEC 9834-8:2005, and related standards.

### UUID Format

A UUID is written as a sequence of lower-case hexadecimal digits, in several groups separated by hyphens, specifically:

- A group of 8 digits
- Three groups of 4 digits
- A group of 12 digits

For a total of 32 digits representing the 128 bits.

### Example UUID

```markdown
a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11
```

### Alternative Input Formats

PostgreSQL also accepts the following alternative forms for input:

- Use of upper-case digits
- The standard format surrounded by braces
- Omitting some or all hyphens
- Adding a hyphen after any group of four digits

### Examples of Alternative Input Formats

```markdown
A0EEBC99-9C0B-4EF8-BB6D-6BB9BD380A11
{a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11}
a0eebc999c0b4ef8bb6d6bb9bd380a11
a0ee-bc99-9c0b-4ef8-bb6d-6bb9-bd38-0a11
{a0eebc99-9c0b4ef8-bb6d6bb9-bd380a11}
```

### Output Format

Output is always in the standard form.

### Generating a UUID in PostgreSQL

See Section 9.14 for how to generate a UUID in PostgreSQL.

### Usage

Here is an example of creating a table with a `uuid` column and inserting a UUID value:

```sql
CREATE TABLE users (
    id uuid PRIMARY KEY,
    name varchar(50)
);

INSERT INTO users (id, name)
VALUES ('a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11', 'John Doe');
```

You can also use the `uuid_generate_v4()` function to generate a random UUID:

```sql
INSERT INTO users (id, name)
VALUES (uuid_generate_v4(), 'Jane Doe');
```

Note: The `uuid_generate_v4()` function is part of the `uuid-ossp` module, which must be installed separately.
