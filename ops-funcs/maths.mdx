## Mathematical Functions and Operators

PostgreSQL provides a wide range of mathematical functions and operators that can be used to perform various mathematical operations. This documentation covers the available mathematical operators and functions, along with examples and usage.

### Mathematical Operators

The following mathematical operators are available for the standard numeric types:

#### Arithmetic Operators

- **Addition**: `numeric_type + numeric_type → numeric_type`  
  Example: `2 + 3 → 5`

- **Unary Plus**: `+ numeric_type → numeric_type`  
  Example: `+ 3.5 → 3.5`

- **Subtraction**: `numeric_type - numeric_type → numeric_type`  
  Example: `2 - 3 → -1`

- **Negation**: `- numeric_type → numeric_type`  
  Example: `- (-4) → 4`

- **Multiplication**: `numeric_type * numeric_type → numeric_type`  
  Example: `2 * 3 → 6`

- **Division**: `numeric_type / numeric_type → numeric_type`  
  Example: `5.0 / 2 → 2.5000000000000000`

- **Modulo**: `numeric_type % numeric_type → numeric_type`  
  Example: `5 % 4 → 1`

- **Exponentiation**: `numeric_type ^ numeric_type → numeric_type`  
  Example: `2 ^ 3 → 8`

- **Square Root**: `|/ double precision → double precision`  
  Example: `|/ 25.0 → 5`

- **Cube Root**: `||/ double precision → double precision`  
  Example: `||/ 64.0 → 4`

- **Absolute Value**: `@ numeric_type → numeric_type`  
  Example: `@ -5.0 → 5.0`

#### Bitwise Operators

- **Bitwise AND**: `integral_type & integral_type → integral_type`  
  Example: `91 & 15 → 11`

- **Bitwise OR**: `integral_type | integral_type → integral_type`  
  Example: `32 | 3 → 35`

- **Bitwise Exclusive OR**: `integral_type # integral_type → integral_type`  
  Example: `17 # 5 → 20`

- **Bitwise NOT**: `~ integral_type → integral_type`  
  Example: `~1 → -2`

- **Bitwise Shift Left**: `integral_type << integer → integral_type`  
  Example: `1 << 4 → 16`

- **Bitwise Shift Right**: `integral_type >> integer → integral_type`  
  Example: `8 >> 2 → 2`

### Mathematical Functions

The following mathematical functions are available:

#### General Mathematical Functions

- **Absolute Value**: `abs(numeric_type) → numeric_type`  
  Example: `abs(-17.4) → 17.4`

  ```sql
  SELECT abs(-17.4);
  ```

- **Cube Root**: `cbrt(double precision) → double precision`  
  Example: `cbrt(64.0) → 4`

  ```sql
  SELECT cbrt(64.0);
  ```

- **Ceiling**: `ceil(numeric) → numeric`  
  Example: `ceil(42.2) → 43`

  ```sql
  SELECT ceil(42.2);
  ```

- **Ceiling (Alternative)**: `ceiling(numeric) → numeric`  
  Example: `ceiling(95.3) → 96`

  ```sql
  SELECT ceiling(95.3);
  ```

- **Degrees**: `degrees(double precision) → double precision`  
  Example: `degrees(0.5) → 28.64788975654116`

  ```sql
  SELECT degrees(0.5);
  ```

- **Division**: `div(y numeric, x numeric) → numeric`  
  Example: `div(9, 4) → 2`

  ```sql
  SELECT div(9, 4);
  ```

- **Exponential**: `exp(numeric) → numeric`  
  Example: `exp(1.0) → 2.7182818284590452`

  ```sql
  SELECT exp(1.0);
  ```

- **Factorial**: `factorial(bigint) → numeric`  
  Example: `factorial(5) → 120`

  ```sql
  SELECT factorial(5);
  ```

- **Floor**: `floor(numeric) → numeric`  
  Example: `floor(42.8) → 42`

  ```sql
  SELECT floor(42.8);
  ```

- **Greatest Common Divisor**: `gcd(numeric_type, numeric_type) → numeric_type`  
  Example: `gcd(1071, 462) → 21`

  ```sql
  SELECT gcd(1071, 462);
  ```

- **Least Common Multiple**: `lcm(numeric_type, numeric_type) → numeric_type`  
  Example: `lcm(1071, 462) → 23562`

  ```sql
  SELECT lcm(1071, 462);
  ```

- **Natural Logarithm**: `ln(numeric) → numeric`  
  Example: `ln(2.0) → 0.6931471805599453`

  ```sql
  SELECT ln(2.0);
  ```

- **Base 10 Logarithm**: `log(numeric) → numeric`  
  Example: `log(100) → 2`

  ```sql
  SELECT log(100);
  ```

- **Base 10 Logarithm (Alternative)**: `log10(numeric) → numeric`  
  Example: `log10(1000) → 3`

  ```sql
  SELECT log10(1000);
  ```

- **Logarithm to Base**: `log(b numeric, x numeric) → numeric`  
  Example: `log(2.0, 64.0) → 6.0000000000000000`

  ```sql
  SELECT log(2.0, 64.0);
  ```

- **Minimum Scale**: `min_scale(numeric) → integer`  
  Example: `min_scale(8.4100) → 2`

  ```sql
  SELECT min_scale(8.4100);
  ```

- **Modulo**: `mod(y numeric_type, x numeric_type) → numeric_type`  
  Example: `mod(9, 4) → 1`

  ```sql
  SELECT mod(9, 4);
  ```

- **Pi**: `pi() → double precision`  
  Example: `pi() → 3.141592653589793`

  ```sql
  SELECT pi();
  ```

- **Power**: `power(a numeric, b numeric) → numeric`  
  Example: `power(9, 3) → 729`

  ```sql
  SELECT power(9, 3);
  ```
